Typography

There are a couple of things you need to set up before you begin a project.

  • base font family
  • base font sizes
  • base line heights and text heirarchy

The best way to do this is sit down with the designer and go through the text heirarchy template with them.

Below is a list of helper classes that relate to Typography

Font sizes

Notes for Heading sizes, text sizes and sub headings

Headings

These heading classes are helpful when you have a heading that is semantically correct but the default size is not the size you want.

code 


    <h1> H1 Heading 1 </h1>
    <h1 class='h2'> H1 Heading with a class of H2 </h1>
    <h2> H2 Heading </h2>
    <h2 class='h4'> H2 Heading with a class of H4 </h2>
    <h4 class='h4'> H4 Heading  </h4>

hello world
